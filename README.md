# JavaFX_Oracle_Project 🐥


## 1. 프로젝트 소개 🚀

- 개요 : EZEN ACADEMY 2개월차 2번째 프로젝트
- 개발 주제 : Java, JavaFX, Oracle 를 활용해 GUI 구현 및 DB연동
- 주제 : 성적관리 프로그램
- 기간 : 2022.06.13 ~ 2022.06.28
- 요약 : 학생 이름과 각 과목의 점수를 입력하면, 총점, 평균, 등급이 자동 계산되어 입력되고 DB에 저장하는 프로그램
